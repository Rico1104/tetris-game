# Rico's Tetris Game

A simple but fun Tetris game built with HTML5, CSS, and JavaScript.

## How to Play

- Use Arrow Keys (←, →) to move the piece left or right
- Up Arrow (↑) to rotate the piece
- Down Arrow (↓) to soft drop
- Spacebar to hard drop

## Features

- Score system
- Level progression
- Next piece preview
- Pause functionality
